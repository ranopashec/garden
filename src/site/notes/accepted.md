# Разрешенные пользователи

Список никнеймов (username) пользователей Telegram, которым разрешен доступ к приватным статьям.

Формат: один username на строку, без символа @ в начале.

Пример:
```
username1
username2
another_user
```

Примечание:
- Пустые строки игнорируются
- Строки, начинающиеся с #, считаются комментариями и игнорируются
- Username проверяется без учета регистра
